## 1. What is Document Categorization?
Grouping documents into categories based on their content.

## 2. Why is it important in IDMS?
Helps in organizing, faster retrieval, and better search.

## 3. Categories in our project
Example:

* Invoices
* Reports
* Certificates
* ID Proofs
* Bills
* Contracts

## 4. Categorization Methods
* Manual categorization
* Rule-based categorization
* AI/ML based categorization

## 5. Challenges
* Mixed content documents
* Incorrect classification
* Similar document structures

## 6. Future improvements
* AI-based smart classification
* Auto-tagging
