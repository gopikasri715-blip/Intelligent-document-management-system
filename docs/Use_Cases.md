# Use Cases

## Actor: User

### Use Case 1: Upload Document

* User selects a document.
* System uploads and stores the document.
* Metadata is saved in the database.

### Use Case 2: Search Document

* User enters a keyword.
* System searches document content.
* Matching documents are displayed.

### Use Case 3: View Document

* User selects a document.
* System displays document details.

### Use Case 4: Download Document

* User chooses a document.
* System provides the download.

## Actor: Admin

### Use Case 1: Manage Documents

* View all uploaded documents.
* Delete unnecessary documents.

### Use Case 2: Monitor System

* View document statistics.
* Track document activity.
