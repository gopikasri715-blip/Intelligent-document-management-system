# Week 2–3 Internship Report

**Project:** Intelligent Document Management System (IDMS)
**Week:** 2 – 3
**Duration:** 21 June 2026 – 04 July 2026

## Overview

During Week 2 and Week 3 of the internship, the research and design phase of the Intelligent Document Management System was completed. The main focus was on studying OCR technology, analyzing document search mechanisms, researching security practices, defining API structures, planning database relationships, and finalizing the technical architecture required for implementation.

## Work Completed

### 1. OCR Research

* Studied Optical Character Recognition (OCR) technology
* Understood OCR workflow and its role in IDMS
* Analyzed OCR challenges and supported file formats

### 2. OCR Tools Comparison

* Compared Tesseract OCR, Google Vision OCR, and AWS Textract
* Evaluated based on speed, accuracy, and integration
* Selected Tesseract OCR for project implementation

### 3. Document Search Research

* Studied different document retrieval methods
* Planned keyword search, metadata search, and full-text search
* Defined search workflow

### 4. Security and Access Control

* Researched authentication and authorization methods
* Planned role-based access control
* Studied encryption and secure document handling

### 5. API Design

Defined APIs for:

* User authentication
* Document upload
* OCR text extraction
* Search and retrieval
* Category management

### 6. Database Relationships

Designed and documented relationships between:

* Users
* Documents
* Categories
* Activity logs

### 7. Technical Architecture

Defined the complete technical stack:

* Frontend: HTML, CSS, JavaScript
* Backend: Flask
* Database: SQLite / MySQL
* OCR Engine: Tesseract OCR

### 8. Design Conclusion

Finalized the complete system design and prepared implementation-ready documentation.
